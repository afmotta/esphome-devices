# esphome-devices

ESPHome firmware and configuration for a three-floor residence's home automation:
CAN-bus wall buttons and room sensors, lighting control, and multi-zone climate
control (radiant floor + fancoil, with ventilation). The whole system is pre-live —
hardware is standardized and the firmware is built, but nothing is physically
deployed yet.

## Maintenance & Support Guide

**[afmotta.github.io/esphome-devices](https://afmotta.github.io/esphome-devices/)** —
the practical guide for keeping this system running: everyday monitoring, diagnosing
common problems, and step-by-step hardware-replacement procedures. Written for
someone who doesn't need to know the codebase — a technician, a family member, or
future-you standing next to a dead piece of hardware having scanned a QR code.
Available in [English](https://afmotta.github.io/esphome-devices/) and
[Italiano](https://afmotta.github.io/esphome-devices/it/).
