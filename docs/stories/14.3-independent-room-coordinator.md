# Story 14.3: Independent Room Boost Coordinator

**Epic:** 14 - Per-Room Fancoil Boost Control  
**Date:** December 26, 2025  
**Status:** Ready  
**Story Points:** 3  
**Priority:** High

---

## User Story

As a **system owner with independent room control**,  
I want **the coordinator to evaluate each room independently**,  
So that **only the room needing boost activates its fancoil and radiant override**.

---

## Story Context

**Current Logic (Epic 13):**
```cpp
max_delta = max(soggiorno_delta, cucina_delta);
if (max_delta > threshold) {
  // Activate BOTH rooms
  slow_pwm_soggiorno.set_level(1.0);
  slow_pwm_cucina.set_level(1.0);
  pid_fancoil_soggiorno.set_mode("COOL");
  pid_fancoil_cucina.set_mode("COOL");
}
```

**Target Logic (Epic 14):**
```cpp
// Evaluate Soggiorno independently
if (soggiorno_delta > threshold && !soggiorno_boost_active) {
  soggiorno_boost_active = true;
  slow_pwm_soggiorno.set_level(1.0);  // Only Soggiorno
  pid_fancoil_soggiorno.set_mode("COOL");
}

// Evaluate Cucina independently  
if (cucina_delta > threshold && !cucina_boost_active) {
  cucina_boost_active = true;
  slow_pwm_cucina.set_level(1.0);  // Only Cucina
  pid_fancoil_cucina.set_mode("COOL");
}
```

---

## Acceptance Criteria

### Core Logic

1. **Independent Evaluation:**
   - [ ] Soggiorno evaluated against threshold independently
   - [ ] Cucina evaluated against threshold independently
   - [ ] Each room uses its own delta sensor

2. **Independent Activation:**
   - [ ] When Soggiorno enters boost: only slow_pwm_soggiorno → 100%
   - [ ] When Soggiorno enters boost: only pid_fancoil_soggiorno → COOL
   - [ ] Cucina remains unaffected (and vice versa)

3. **Independent Deactivation:**
   - [ ] When Soggiorno exits boost: pid_fancoil_soggiorno → OFF
   - [ ] Soggiorno radiant PID resumes control
   - [ ] Cucina remains in its current state

4. **Independent Timing:**
   - [ ] Each room has its own 10-minute minimum time-in-state
   - [ ] Soggiorno can exit while Cucina remains in boost

### Winter Mode

5. **Season Handling:**
   - [ ] Both rooms exit boost when summer_mode = false
   - [ ] Both fancoils turn OFF
   - [ ] Radiant PIDs resume control

### Zone Demand

6. **Fancoil Pump Update:**
   - [ ] Update `ground_floor_fancoil_any_zone_open` in ground-floor.yaml
   - [ ] New logic: `soggiorno_boost_active OR cucina_boost_active OR pid_fancoil_*_is_active`

## Technical Implementation

### Refactored Coordinator Interval

```yaml
interval:
  - interval: 30s
    then:
      - lambda: |-
          // ======== COMMON SETUP ========
          if (!id(summer_mode).state) {
            // Winter: exit boost for both rooms
            if (id(soggiorno_boost_active)) {
              id(soggiorno_boost_active) = false;
              auto call = id(pid_fancoil_soggiorno).make_call();
              call.set_mode("OFF");
              call.perform();
            }
            if (id(cucina_boost_active)) {
              id(cucina_boost_active) = false;
              auto call = id(pid_fancoil_cucina).make_call();
              call.set_mode("OFF");
              call.perform();
            }
            return;
          }
          
          float threshold = id(effective_fancoil_boost_threshold).state;
          float hysteresis = threshold / 2.0;
          uint32_t now = millis();
          uint32_t min_time_ms = ${min_time_in_state_seconds} * 1000;
          
          if (std::isnan(threshold)) {
            ESP_LOGW("coordinator", "Invalid threshold, skipping");
            return;
          }
          
          // ======== SOGGIORNO EVALUATION ========
          {
            float delta = id(soggiorno_temp_delta).state;
            bool can_change = (id(soggiorno_last_mode_change) == 0 || 
                              (now - id(soggiorno_last_mode_change)) >= min_time_ms);
            
            if (!std::isnan(delta) && can_change) {
              // Activation
              if (!id(soggiorno_boost_active) && delta > threshold) {
                id(soggiorno_boost_active) = true;
                id(soggiorno_last_mode_change) = now;
                id(slow_pwm_soggiorno).set_level(1.0);
                auto call = id(pid_fancoil_soggiorno).make_call();
                call.set_mode("COOL");
                call.perform();
                ESP_LOGI("coordinator", "SOGGIORNO: Boost ACTIVATED (delta=%.1f > threshold=%.1f)", delta, threshold);
              }
              // Deactivation
              if (id(soggiorno_boost_active) && delta < hysteresis) {
                id(soggiorno_boost_active) = false;
                id(soggiorno_last_mode_change) = now;
                auto call = id(pid_fancoil_soggiorno).make_call();
                call.set_mode("OFF");
                call.perform();
                ESP_LOGI("coordinator", "SOGGIORNO: Boost DEACTIVATED (delta=%.1f < hysteresis=%.1f)", delta, hysteresis);
              }
            }
          }
          
          // ======== CUCINA EVALUATION ========
          {
            float delta = id(cucina_temp_delta).state;
            bool can_change = (id(cucina_last_mode_change) == 0 || 
                              (now - id(cucina_last_mode_change)) >= min_time_ms);
            
            if (!std::isnan(delta) && can_change) {
              // Activation
              if (!id(cucina_boost_active) && delta > threshold) {
                id(cucina_boost_active) = true;
                id(cucina_last_mode_change) = now;
                id(slow_pwm_cucina).set_level(1.0);
                auto call = id(pid_fancoil_cucina).make_call();
                call.set_mode("COOL");
                call.perform();
                ESP_LOGI("coordinator", "CUCINA: Boost ACTIVATED (delta=%.1f > threshold=%.1f)", delta, threshold);
              }
              // Deactivation
              if (id(cucina_boost_active) && delta < hysteresis) {
                id(cucina_boost_active) = false;
                id(cucina_last_mode_change) = now;
                auto call = id(pid_fancoil_cucina).make_call();
                call.set_mode("OFF");
                call.perform();
                ESP_LOGI("coordinator", "CUCINA: Boost DEACTIVATED (delta=%.1f < hysteresis=%.1f)", delta, hysteresis);
              }
            }
          }
```
