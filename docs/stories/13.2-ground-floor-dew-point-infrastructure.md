# Story 13.2: Ground Floor Dew Point Infrastructure

**Epic:** 13 - Ground Floor Hybrid Radiant-Fancoil Cooling  
**Date:** December 26, 2025  
**Status:** Ready for Review  
**Story Points:** 2  
**Priority:** High (Safety Critical)

## User Story

As a **system owner implementing radiant cooling**,  
I want **per-room dew point calculations and a floor-wide maximum aggregation**,  
So that **I can protect against condensation by limiting supply water temperature**.

---

## Story Context

**Existing System Integration:**

- **Reference:** First floor already has dew point infrastructure in `first-floor.yaml`
- **Component:** `dew_point_sensor.yaml` exists and calculates dew point from temp + humidity
- **Pattern:** `sensor.combination` with `type: max` aggregates room dew points
- **Touch points:**
  - `components/rooms/ground_floor/soggiorno.yaml` - Add dew point sensor
  - `components/rooms/ground_floor/cucina.yaml` - Add dew point sensor
  - `components/rooms/ground_floor/anticamera.yaml` - Add dew point sensor
  - `components/rooms/ground_floor/ground-floor.yaml` - Add max aggregation

**First Floor Reference Pattern:**

```yaml
# From components/rooms/first_floor/first-floor.yaml
sensor:
  - platform: combination
    id: first_floor_max_dew_point
    name: "Max Dew Point Primo piano"
    type: max
    sources:
      - source: camera_ospiti_dew_point
      - source: camera_nord_dew_point
      - source: camera_padronale_dew_point
      - source: camera_sud_dew_point
    on_value:
      then:
        - climate.control:
            id: pid_mixing_valve_first_floor_radiant
            target_temperature: !lambda "return (x + ${safe_margin});"
```

**Dew Point Calculation (Magnus-Tetens formula):**

```yaml
# From components/dew_point_sensor.yaml
lambda: |-
  const float alpha = 6.112;
  const float beta = 17.62;
  const float lambda = 243.12;
  float RH = id(${humidity_sensor}).raw_state;
  float T = id(${temperature_sensor}).raw_state;
  float H = log(RH / 100) + beta * T / (lambda + T);
  return (lambda) * H / (beta - H);
```

### Ground Floor Aggregation

**In ground-floor.yaml:**
```yaml
defaults:
  safe_margin: 1.0  # °C above dew point

sensor:
  - platform: combination
    id: ground_floor_max_dew_point
    name: "Max Dew Point Piano Terra"
    type: max
    unit_of_measurement: "°C"
    device_class: "temperature"
    state_class: "measurement"
    sources:
      - source: soggiorno_dew_point
      - source: cucina_dew_point
      - source: anticamera_dew_point
```
