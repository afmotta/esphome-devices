## Test Procedures

### Test 1: Normal Mode Operation

**Setup:**
1. Enable cooling mode (summer_mode = true)
2. Set all rooms to setpoints above current temp (ensure cooling demand)
3. Ensure delta < threshold (normal mode)

**Procedure:**
1. Wait 5 minutes for system to stabilize
2. Verify radiant PIDs show COOLING action
3. Verify fancoil PIDs show OFF
4. Verify mixing pump is running
5. Verify diagnostic sensor shows "Radiant Only"

**Expected Result:** System operates in normal radiant cooling mode

---

### Test 2: Boost Activation

**Setup:**
1. System in normal mode
2. Identify method to increase delta > threshold
   - Option A: Lower setpoint significantly
   - Option B: Wait for hot weather
   - Option C: Use heat source in room (carefully)

**Procedure:**
1. Create condition where delta > threshold (e.g., 4°C)
2. Wait for coordinator interval (30 seconds)
3. Verify mode changes to "Fancoil Boost"
4. Verify radiant zones at 100%
5. Verify fancoil PIDs in COOL mode

**Expected Result:** System transitions to boost mode

---

### Test 3: Boost Deactivation with Hysteresis

**Setup:**
1. System in boost mode
2. Delta currently > threshold

**Procedure:**
1. Allow cooling to reduce room temperature
2. When delta approaches hysteresis (threshold/2), observe
3. Verify boost remains active until delta < hysteresis
4. Verify minimum time-in-state prevents premature deactivation

**Expected Result:** Boost deactivates only after hysteresis threshold crossed AND minimum time elapsed

---

### Test 4: Dew Point Protection

**Setup:**
1. System in cooling mode
2. Method to increase humidity:
   - Hot shower in bathroom (if near monitored rooms)
   - Or simulate via HA sensor override

**Procedure:**
1. Note current supply temp minimum
2. Increase humidity in monitored rooms
3. Observe ground_floor_max_dew_point increase
4. Verify supply temp minimum increases accordingly
5. Verify supply temperature respects new minimum

**Expected Result:** Supply temperature minimum adjusts to protect against condensation

---

### Test 5: HA Unavailability

**Setup:**
1. System in normal operation
2. Note current threshold value from HA

**Procedure:**
1. Stop Home Assistant service
2. Wait 5 minutes
3. Verify ESPHome logs show fallback threshold being used
4. Verify system continues cooling operation
5. Restart Home Assistant
6. Verify HA threshold value restored

**Expected Result:** System operates autonomously during HA outage

---

### Test 6: Full Day Operation

**Setup:**
1. System fully deployed
2. Normal summer day weather

**Procedure:**
1. Monitor system for 24 hours
2. Log any mode transitions
3. Check for unexpected behavior
4. Verify no oscillation
5. Verify resource usage stable

**Expected Result:** Stable operation over extended period
