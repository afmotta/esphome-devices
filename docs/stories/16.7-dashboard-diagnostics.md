# Story 16.7: Dashboard & Diagnostics

**Epic:** 16 - First Floor MEV Intelligent Control  
**Date:** January 15, 2026  
**Status:** Ready  
**Story Points:** 1  
**Priority:** Medium

---

## User Story

As a **homeowner**,  
I want **a dashboard card showing MEV status, current stage, and diagnostic information**,  
So that **I can understand what the system is doing and troubleshoot if needed**.

---

## Story Context

All the control logic is now in place. This story creates a dashboard view that shows:
- Current humidity stage
- Air quality readings and demand
- Equipment states (fan, dehumidifier, cooling)
- Pump protection timer status
- Override controls

---

## Acceptance Criteria

### Functional Requirements

1. **Status Overview:**
   - [ ] Current humidity stage (Normal/Elevated/High/Critical)
   - [ ] Max humidity reading and source room
   - [ ] Max CO₂ and IAQ readings

2. **Equipment Status:**
   - [ ] Fan speed (current commanded value)
   - [ ] Dehumidifier state (ON/OFF)
   - [ ] Cooling pump state (ON/OFF)
   - [ ] Summer mode indicator

3. **Demand Breakdown:**
   - [ ] Air quality demand (%)
   - [ ] Humidity stage demand (%)
   - [ ] Driving signal indicator (CO₂ or IAQ)

4. **Timer Status:**
   - [ ] Pump min_on timer (remaining time if active)
   - [ ] Pump min_off timer (remaining time if active)

5. **Manual Controls:**
   - [ ] Manual stage override (input_select)
   - [ ] Alarm acknowledgment (if active)

### Technical Requirements

6. **Dashboard location:** Home Assistant Overview or dedicated Climate view
7. **Responsive layout:** Works on mobile and desktop
8. **Real-time updates:** No page refresh needed

---

## Technical Implementation

### Dashboard Card (Lovelace YAML)

**Option 1: Entities Card (Simple)**
```yaml
type: entities
title: VMC Primo Piano
entities:
  # Status Section
  - entity: input_select.mev_humidity_stage
    name: Humidity Stage
    icon: mdi:water-percent
  - entity: sensor.first_floor_max_humidity
    name: Max Humidity
    icon: mdi:water-percent
  - entity: sensor.first_floor_max_co2
    name: Max CO₂
    icon: mdi:molecule-co2
  - entity: sensor.first_floor_max_iaq
    name: Max IAQ
    icon: mdi:air-filter
  
  # Divider
  - type: divider
  
  # Demand Section
  - entity: sensor.mev_air_quality_demand
    name: AQ Demand
    icon: mdi:fan
  - entity: sensor.mev_humidity_stage_demand
    name: Stage Demand
    icon: mdi:gauge
  - entity: sensor.mev_final_fan_speed
    name: Final Fan Speed
    icon: mdi:fan-speed-3
  
  # Divider
  - type: divider
  
  # Equipment Section
  - entity: number.first_floor_mev_fan_speed
    name: Fan Speed
  - entity: switch.first_floor_mev_dehumidifier
    name: Dehumidifier
  - entity: switch.first_floor_mev_cooling
    name: Cooling Pump
  - entity: binary_sensor.summer_mode
    name: Summer Mode
  
  # Divider
  - type: divider
  
  # Timer Section
  - entity: timer.mev_cooling_pump_min_on
    name: Pump Min ON
  - entity: timer.mev_cooling_pump_min_off
    name: Pump Min OFF
```

**Option 2: Custom Cards (Advanced)**
```yaml
type: vertical-stack
cards:
  # Header with stage badge
  - type: markdown
    content: |
      ## 🌬️ VMC Primo Piano
      **Stage:** {{ states('input_select.mev_humidity_stage') | title }}
      {% set stage = states('input_select.mev_humidity_stage') %}
      {% if stage == 'critical' %}🔴{% elif stage == 'high' %}🟠{% elif stage == 'elevated' %}🟡{% else %}🟢{% endif %}

  # Gauge cards row
  - type: horizontal-stack
    cards:
      - type: gauge
        entity: sensor.first_floor_max_humidity
        name: Humidity
        min: 30
        max: 90
        severity:
          green: 0
          yellow: 55
          red: 65
      - type: gauge
        entity: sensor.mev_final_fan_speed
        name: Fan Speed
        min: 0
        max: 100
        severity:
          green: 0
          yellow: 50
          red: 80

  # Air quality indicators
  - type: horizontal-stack
    cards:
      - type: sensor
        entity: sensor.first_floor_max_co2
        name: CO₂
        icon: mdi:molecule-co2
        graph: line
      - type: sensor
        entity: sensor.first_floor_max_iaq
        name: IAQ
        icon: mdi:air-filter
        graph: line

  # Equipment status
  - type: glance
    title: Equipment
    entities:
      - entity: switch.first_floor_mev_dehumidifier
        name: Dehumid
      - entity: switch.first_floor_mev_cooling
        name: Cooling
      - entity: binary_sensor.summer_mode
        name: Summer
      - entity: binary_sensor.first_floor_mev_alarm
        name: Alarm

  # Timers (conditional)
  - type: conditional
    conditions:
      - entity: timer.mev_cooling_pump_min_on
        state_not: idle
    card:
      type: entity
      entity: timer.mev_cooling_pump_min_on
      name: Pump Min ON Remaining

  - type: conditional
    conditions:
      - entity: timer.mev_cooling_pump_min_off
        state_not: idle
    card:
      type: entity
      entity: timer.mev_cooling_pump_min_off
      name: Pump Min OFF Remaining

  # Manual override
  - type: entities
    title: Manual Override
    entities:
      - entity: input_select.mev_humidity_stage
        name: Force Stage
```

```yaml
  **Air Quality Demand:** {{ states('sensor.mev_air_quality_demand') }}%

  - IAQ: {{ state_attr('sensor.mev_air_quality_demand', 'iaq_level') }} → {{ state_attr('sensor.mev_air_quality_demand', 'iaq_demand') }}%

```
