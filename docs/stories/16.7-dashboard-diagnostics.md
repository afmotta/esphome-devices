# Story 16.7: Dashboard & Diagnostics

**Epic:** 16 - First Floor MEV Intelligent Control  
**Date:** January 15, 2026  
**Updated:** January 16, 2026  
**Status:** Ready  
**Story Points:** 1  
**Priority:** Medium

---

## User Story

As a **homeowner**,  
I want **a dashboard card showing MEV status, current state, and diagnostic information**,  
So that **I can understand what the system is doing and troubleshoot if needed**.

---

## Story Context

All the control logic is now in place. This story creates a dashboard view that shows:
- Current humidity state (fan_only / dehumidifying / cooling)
- Air quality and humidity readings
- Fan speed demand breakdown (AQ vs humidity)
- Equipment states (fan, dehumidifier, cooling)
- Configuration parameters (humidity bounds, fan thresholds)

---

## Acceptance Criteria

### Functional Requirements

1. **Status Overview:**
   - [ ] Current humidity state (fan_only / dehumidifying / cooling)
   - [ ] Max humidity reading
   - [ ] Max CO₂ and max IAQ readings

2. **Demand Breakdown:**
   - [ ] Air quality demand (%)
   - [ ] Humidity demand (%)
   - [ ] Final fan speed (%)
   - [ ] Humidity rate of change (%/min)

3. **Equipment Status:**
   - [ ] Fan speed (current commanded value)
   - [ ] Dehumidifier state (ON/OFF)
   - [ ] Cooling pump state (ON/OFF)
   - [ ] Summer mode indicator

4. **Configuration Access:**
   - [ ] Humidity upper bound (configurable)
   - [ ] Humidity lower bound (configurable)
   - [ ] Escalation fan threshold (configurable)
   - [ ] Manual state override (input_select)

5. **Alarm Status:**
   - [ ] MEV alarm sensor (from Epic 6)

### Technical Requirements

6. **Dashboard location:** Home Assistant Overview or dedicated Climate view
7. **Card type:** Entities card (simple, standard HA)
8. **Optional:** Gauge card for fan speed visualization

**Option 2: Custom Cards (Advanced)**
```yaml
type: vertical-stack
cards:
  # Header with stage badge
  - type: markdown
    content: |
      ## 🌬️ VMC Primo Piano
      **Stage:** {{ states('input_select.mev_humidity_stage') | title }}
      {% set stage = states('input_select.mev_humidity_stage') %}
      {% if stage == 'critical' %}🔴{% elif stage == 'high' %}🟠{% elif stage == 'elevated' %}🟡{% else %}🟢{% endif %}

  # Gauge cards row
  - type: horizontal-stack
    cards:
      - type: gauge
        entity: sensor.first_floor_max_humidity
        name: Humidity
        min: 30
        max: 90
        severity:
          green: 0
          yellow: 55
          red: 65
      - type: gauge
        entity: sensor.mev_final_fan_speed
        name: Fan Speed
        min: 0
        max: 100
        severity:
          green: 0
          yellow: 50
          red: 80

  # Air quality indicators
  - type: horizontal-stack
    cards:
      - type: sensor
        entity: sensor.first_floor_max_co2
        name: CO₂
        icon: mdi:molecule-co2
        graph: line
      - type: sensor
        entity: sensor.first_floor_max_iaq
        name: IAQ
        icon: mdi:air-filter
        graph: line

  # Equipment status
  - type: glance
    title: Equipment
    entities:
      - entity: switch.first_floor_mev_dehumidifier
        name: Dehumid
      - entity: switch.first_floor_mev_cooling
        name: Cooling
      - entity: binary_sensor.summer_mode
        name: Summer
      - entity: binary_sensor.first_floor_mev_alarm
        name: Alarm

  # Timers (conditional)
  - type: conditional
    conditions:
      - entity: timer.mev_cooling_pump_min_on
        state_not: idle
    card:
      type: entity
      entity: timer.mev_cooling_pump_min_on
      name: Pump Min ON Remaining

  - type: conditional
    conditions:
      - entity: timer.mev_cooling_pump_min_off
        state_not: idle
    card:
      type: entity
      entity: timer.mev_cooling_pump_min_off
      name: Pump Min OFF Remaining

  # Manual override
  - type: entities
    title: Manual Override
    entities:
      - entity: input_select.mev_humidity_stage
        name: Force Stage
```
