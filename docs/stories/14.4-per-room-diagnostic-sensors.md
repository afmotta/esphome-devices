# Story 14.4: Per-Room Diagnostic Sensors

**Epic:** 14 - Per-Room Fancoil Boost Control  
**Date:** December 26, 2025  
**Status:** Ready  
**Story Points:** 1  
**Priority:** Medium

---

## User Story

As a **system owner monitoring room climate**,  
I want **per-room diagnostic sensors in Home Assistant**,  
So that **I can see each room's boost status and mode independently**.

---

## Story Context

**Current Sensors (Shared):**
- `ground_floor_cooling_mode` - "Radiant Only" / "Fancoil Boost"
- `fancoil_boost_active` - Boolean
- `ground_floor_max_delta` - Max delta across rooms
- `ground_floor_time_in_current_mode` - Time since last change

**Target Sensors (Per-Room):**
- `soggiorno_cooling_mode` / `cucina_cooling_mode`
- `soggiorno_fancoil_boost_active` / `cucina_fancoil_boost_active`
- `soggiorno_temp_delta` / `cucina_temp_delta` (from Story 14.1)
- `soggiorno_time_in_mode` / `cucina_time_in_mode`

## Technical Implementation

### Per-Room Diagnostic Sensors

```yaml
binary_sensor:
  - platform: template
    id: soggiorno_fancoil_boost_active
    name: "Soggiorno Fancoil Boost Active"
    device_class: running
    icon: "mdi:fan-alert"
    lambda: "return id(soggiorno_boost_active);"

  - platform: template
    id: cucina_fancoil_boost_active
    name: "Cucina Fancoil Boost Active"
    device_class: running
    icon: "mdi:fan-alert"
    lambda: "return id(cucina_boost_active);"

text_sensor:
  - platform: template
    id: soggiorno_cooling_mode
    name: "Soggiorno Cooling Mode"
    icon: "mdi:thermostat"
    lambda: |-
      if (id(soggiorno_boost_active)) {
        return {"Fancoil Boost"};
      }
      return {"Radiant Only"};
    update_interval: 5s

  - platform: template
    id: cucina_cooling_mode
    name: "Cucina Cooling Mode"
    icon: "mdi:thermostat"
    lambda: |-
      if (id(cucina_boost_active)) {
        return {"Fancoil Boost"};
      }
      return {"Radiant Only"};
    update_interval: 5s

sensor:
  - platform: template
    id: soggiorno_time_in_mode
    name: "Soggiorno Time In Mode"
    unit_of_measurement: "min"
    accuracy_decimals: 0
    icon: "mdi:timer"
    lambda: |-
      if (id(soggiorno_last_mode_change) == 0) return 0.0;
      uint32_t elapsed_ms = millis() - id(soggiorno_last_mode_change);
      return elapsed_ms / 60000.0;
    update_interval: 60s

  - platform: template
    id: cucina_time_in_mode
    name: "Cucina Time In Mode"
    unit_of_measurement: "min"
    accuracy_decimals: 0
    icon: "mdi:timer"
    lambda: |-
      if (id(cucina_last_mode_change) == 0) return 0.0;
      uint32_t elapsed_ms = millis() - id(cucina_last_mode_change);
      return elapsed_ms / 60000.0;
    update_interval: 60s
```
