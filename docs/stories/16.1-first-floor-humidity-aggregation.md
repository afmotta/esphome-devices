**Status:** Done

**Priority:** High

## User Story

As a **MEV control system**,  
I want **a floor-wide maximum humidity reading from all 8 first floor rooms**,  
So that **the humidity cascade state machine can respond to the worst humidity on the floor**.

---

## Story Context

**Existing Pattern:**

- First floor already has `first_floor_max_dew_point` using `sensor.combination` with `type: max`
- Room humidity sensors exist via `room_sensors.yaml` with `${room_slug}_room_humidity_abstracted` pattern
- Max humidity drives the humidity cascade; highest reading determines response level

**Design Decision (from Brainstorming):**

- Include ALL 8 first floor rooms (not just bathrooms)
- Bedroom humidity matters because it impacts radiant cooling effectiveness
- "Worst case wins" ensures no room is neglected

### Implementation

**Add to first-floor.yaml sensor section:**
```yaml
sensor:
  # Max Humidity for MEV Cascade Control
  - platform: combination
    id: first_floor_max_humidity
    name: "First Floor Max Humidity"
    type: max
    unit_of_measurement: "%"
    device_class: humidity
    state_class: "measurement"
    accuracy_decimals: 0
    icon: "mdi:water-percent"
    sources:
      - source: bagno_grande_room_humidity_abstracted
      - source: bagno_ospiti_room_humidity_abstracted
      - source: bagno_padronale_room_humidity_abstracted
      - source: camera_nord_room_humidity_abstracted
      - source: camera_sud_room_humidity_abstracted
      - source: camera_ospiti_room_humidity_abstracted
      - source: camera_padronale_room_humidity_abstracted
      - source: lavanderia_room_humidity_abstracted
```
