# Story 8.5 Tier 1 Migration Plan - First Floor Rooms

**Date:** November 1, 2025  
**Epic:** 8 - Unified State Machine Architecture  
**Phase:** Tier 1 Rollout (First Floor Radiant Rooms)  
**Rooms:** 3 rooms (radiant floor heating only)

---

## Overview

This document details the Tier 1 migration plan for first-floor rooms from Epic 5 to Epic 8 architecture. All first-floor rooms use **radiant floor heating only** (no fancoils), so they require **emergency condition only** (no window condition).

---

## Tier 1 Rooms - Architecture Clarification

### Equipment Configuration
All Tier 1 rooms have:
- ✅ Radiant floor heating with PID control
- ❌ NO fancoil systems
- ❌ NO window sensors (not applicable for radiant-only)

### Epic 8 Components Required
- ✅ `room_emergency_condition.yaml` - Sensor failure detection
- ✅ `room_control_coordinator.yaml` - PID control coordination
- ❌ **NOT** `room_window_condition.yaml` - Only for fancoil rooms

**Rationale:** Window detection is designed for fancoil systems that waste energy when windows open. Radiant floor heating has high thermal inertia and minimal energy waste from open windows, making window detection unnecessary.

```bash

git add components/rooms/first_floor/{room_name}.yaml
git commit -m "Pre-Epic-8: {RoomName} baseline (Epic 5)"

# Create timestamped backup
cp locals/distribuzione-primo-piano.yaml \
   locals/distribuzione-primo-piano.yaml.bak-{room_slug}-$(date +%Y%m%d-%H%M%S)
```

### Step 8: Functional Testing (Post-Deployment)
**Test 1: Emergency Condition (15 minutes)**
1. Note baseline: Check `text_sensor.{room_slug}_coordinator_status` = "Normal (All Clear)"
2. Trigger: Disconnect HA temperature sensor (restart HA integration or mock failure)
3. Wait 3 minutes (emergency timeout)
4. Validate: Coordinator status = "Shutdown: Emergency (Active)"
5. Validate: `climate.pid_radiant_{room_slug}` = OFF
6. Clear: Restore HA temperature sensor
7. Wait 1 minute (recovery timeout)
8. Validate: Coordinator status = "Normal (All Clear)"
9. Manual resume: Set PID back to previous mode via HA

**Test 2: Coordinator Observability (5 minutes)**
1. Check `text_sensor.{room_slug}_emergency_state` exists and shows "Normal"
2. Check `text_sensor.{room_slug}_coordinator_status` exists and updates every 5s
3. Verify HA history graphs show clean state transitions

### Step 9: Overnight Soak Test (12+ hours)
1. Monitor ESPHome logs for errors (none expected)
2. Check temperature control quality (±1°C from baseline)
3. Verify coordinator status remains stable
4. Confirm no unexpected state transitions

### Step 10: Sign-Off
- [ ] Functional tests passed (emergency trigger/clear)
- [ ] Overnight soak test passed (12h+ stable)
- [ ] Climate quality maintained (±1°C)
- [ ] No errors in logs
- [ ] Coordinator status accurate
- ✅ Proceed to next room

## Risk Mitigation

**Risk:** All 3 rooms on same ESP32 device (distribuzione-primo-piano)

**Mitigation:**
- Stagger migrations 1 room/day
- Each room independently validated before next migration
- Device-level rollback available if systematic issues detected
- Monitor ESP32 free heap during rollout (current rooms already stable)

**Stop-Gate Criteria:**
- ❌ Any room requires rollback after soak test
- ❌ Climate quality degrades >2°C in any room
- ❌ Performance issues (memory leaks, delays)
- ❌ Multiple errors detected on device

## HA Automation Template

```yaml
# Add one automation per room migrated
automation:
  - id: epic8_coordinator_resume_bagno_grande
    alias: "Epic 8: Resume Bagno Grande Climate"
    trigger:
      - platform: state
        entity_id: text_sensor.bagno_grande_coordinator_status
        to: "Normal (All Clear)"
        for: "00:00:30"
    condition:
      - condition: state
        entity_id: climate.pid_radiant_bagno_grande
        state: "off"
    action:
      - service: climate.set_hvac_mode
        target:
          entity_id: climate.pid_radiant_bagno_grande
        data:
          hvac_mode: "heat"  # Radiant is typically heat-only
      - service: system_log.write
        data:
          message: "Epic 8: Resumed Bagno Grande climate control"
          level: info

  # Repeat for camera_padronale and camera_nord
```
