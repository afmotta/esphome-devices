# Story 16.6: HA Automation Implementation

**Epic:** 16 - First Floor MEV Intelligent Control  
**Date:** January 15, 2026  
**Status:** Ready  
**Story Points:** 3  
**Priority:** High

---

## User Story

As a **climate control system**,  
I want **a main automation that controls MEV equipment based on the humidity stage and air quality demand**,  
So that **fan speed, dehumidifier, and cooling pump respond correctly to current conditions**.

---

## Story Context

This is the central automation that ties all the pieces together:

- **Story 16.1:** Provides `sensor.first_floor_max_humidity`
- **Story 16.2:** Provides `sensor.mev_air_quality_demand`
- **Story 16.3:** Provides `input_select.mev_humidity_stage` + transition automations
- **Story 16.4:** Provides pump protection timers
- **Story 16.5:** Provides all helper entities

This automation reads the current humidity stage and commands:
- Fan speed → `number.first_floor_mev_fan_speed`
- Dehumidifier → `switch.first_floor_mev_dehumidifier`
- Cooling pump → `switch.first_floor_mev_cooling` (with protection)

---

## Acceptance Criteria

### Functional Requirements

1. **Trigger on:**
   - [ ] Humidity stage change (`input_select.mev_humidity_stage`)
   - [ ] Air quality demand change (`sensor.mev_air_quality_demand`)
   - [ ] Periodic fallback (every 1 minute)

2. **Stage 0 (Normal) Actions:**
   - [ ] Fan speed = AQ demand
   - [ ] Dehumidifier OFF
   - [ ] Cooling pump OFF (with protection)

3. **Stage 1 (Elevated) Actions:**
   - [ ] Fan speed = min(AQ demand + 20%, 100%)
   - [ ] Dehumidifier OFF
   - [ ] Cooling pump OFF

4. **Stage 2 (High) Actions:**
   - [ ] Fan speed = max(70%, AQ demand)
   - [ ] Dehumidifier ON
   - [ ] Cooling pump OFF

5. **Stage 3 (Critical) Actions:**
   - [ ] Fan speed = max(90%, AQ demand)
   - [ ] Dehumidifier ON
   - [ ] Cooling pump ON (if Summer Mode AND min_off idle)

6. **Pump Protection:**
   - [ ] Start only if min_off timer is idle
   - [ ] Stop only if min_on timer is idle
   - [ ] Start min_on timer when pump turns ON
   - [ ] Start min_off timer when pump turns OFF

### Technical Requirements

7. **Automation mode:** `restart` (new triggers restart the automation)
8. **Error handling:** Graceful if any entity unavailable
9. **Logging:** Optional debug notifications during commissioning

## Technical Implementation

### Files to Create (in HA)

| File | Change |
|------|--------|
| `automations.yaml` or HA UI | Main MEV control automation |

### Main Automation

```yaml
automation:
  - alias: "MEV First Floor Intelligent Control"
    id: mev_first_floor_intelligent_control
    description: "Controls MEV fan, dehumidifier, and cooling based on humidity stage and air quality"
    mode: restart
    
    trigger:
      # Stage changes
      - platform: state
        entity_id: input_select.mev_humidity_stage
      # Air quality demand changes
      - platform: state
        entity_id: sensor.mev_air_quality_demand
      # Periodic fallback (ensures state is correct even if triggers missed)
      - platform: time_pattern
        minutes: "/1"
    
    variables:
      # Current humidity stage
      stage: "{{ states('input_select.mev_humidity_stage') }}"
      # Air quality fan demand (20-90%)
      aq_demand: "{{ states('sensor.mev_air_quality_demand') | int(20) }}"
      # Summer mode flag
      summer_mode: "{{ is_state('binary_sensor.summer_mode', 'on') }}"
      # Timer states
      min_on_idle: "{{ is_state('timer.mev_cooling_pump_min_on', 'idle') }}"
      min_off_idle: "{{ is_state('timer.mev_cooling_pump_min_off', 'idle') }}"
      # Current pump state
      pump_on: "{{ is_state('switch.first_floor_mev_cooling', 'on') }}"
    
    action:
      - choose:
          # =========================================
          # STAGE 3: CRITICAL
          # Fan ≥90%, Dehumidifier ON, Cooling ON (summer only, with protection)
          # =========================================
          - conditions:
              - condition: template
                value_template: "{{ stage == 'critical' }}"
            sequence:
              # Set fan speed
              - service: number.set_value
                target:
                  entity_id: number.first_floor_mev_fan_speed
                data:
                  value: "{{ [90, aq_demand] | max }}"
              
              # Turn on dehumidifier
              - service: switch.turn_on
                target:
                  entity_id: switch.first_floor_mev_dehumidifier
              
              # Cooling pump control (with protection)
              - if:
                  - condition: template
                    value_template: "{{ summer_mode and min_off_idle and not pump_on }}"
                then:
                  - service: switch.turn_on
                    target:
                      entity_id: switch.first_floor_mev_cooling
                  - service: timer.start
                    target:
                      entity_id: timer.mev_cooling_pump_min_on
          
          # =========================================
          # STAGE 2: HIGH
          # Fan ≥70%, Dehumidifier ON, Cooling OFF
          # =========================================
          - conditions:
              - condition: template
                value_template: "{{ stage == 'high' }}"
            sequence:
              # Set fan speed
              - service: number.set_value
                target:
                  entity_id: number.first_floor_mev_fan_speed
                data:
                  value: "{{ [70, aq_demand] | max }}"
              
              # Turn on dehumidifier
              - service: switch.turn_on
                target:
                  entity_id: switch.first_floor_mev_dehumidifier
              
              # Turn off cooling pump (with protection)
              - if:
                  - condition: template
                    value_template: "{{ min_on_idle and pump_on }}"
                then:
                  - service: switch.turn_off
                    target:
                      entity_id: switch.first_floor_mev_cooling
                  - service: timer.start
                    target:
                      entity_id: timer.mev_cooling_pump_min_off
          
          # =========================================
          # STAGE 1: ELEVATED
          # Fan = AQ + 20%, Dehumidifier OFF, Cooling OFF
          # =========================================
          - conditions:
              - condition: template
                value_template: "{{ stage == 'elevated' }}"
            sequence:
              # Set fan speed (AQ demand + 20%, capped at 100%)
              - service: number.set_value
                target:
                  entity_id: number.first_floor_mev_fan_speed
                data:
                  value: "{{ [aq_demand + 20, 100] | min }}"
              
              # Turn off dehumidifier
              - service: switch.turn_off
                target:
                  entity_id: switch.first_floor_mev_dehumidifier
              
              # Turn off cooling pump (with protection)
              - if:
                  - condition: template
                    value_template: "{{ min_on_idle and pump_on }}"
                then:
                  - service: switch.turn_off
                    target:
                      entity_id: switch.first_floor_mev_cooling
                  - service: timer.start
                    target:
                      entity_id: timer.mev_cooling_pump_min_off
          
          # =========================================
          # STAGE 0: NORMAL (default)
          # Fan = AQ demand, Dehumidifier OFF, Cooling OFF
          # =========================================
          - conditions:
              - condition: template
                value_template: "{{ stage == 'normal' }}"
            sequence:
              # Set fan speed to AQ demand
              - service: number.set_value
                target:
                  entity_id: number.first_floor_mev_fan_speed
                data:
                  value: "{{ aq_demand }}"
              
              # Turn off dehumidifier
              - service: switch.turn_off
                target:
                  entity_id: switch.first_floor_mev_dehumidifier
              
              # Turn off cooling pump (with protection)
              - if:
                  - condition: template
                    value_template: "{{ min_on_idle and pump_on }}"
                then:
                  - service: switch.turn_off
                    target:
                      entity_id: switch.first_floor_mev_cooling
                  - service: timer.start
                    target:
                      entity_id: timer.mev_cooling_pump_min_off
```

### Equipment Control Summary

| Stage | Fan Speed | Dehumidifier | Cooling Pump |
|-------|-----------|--------------|--------------|
| Normal | AQ demand (20-90%) | OFF | OFF |
| Elevated | AQ + 20% (capped 100%) | OFF | OFF |
| High | max(70%, AQ) | ON | OFF |
| Critical | max(90%, AQ) | ON | ON (summer + protection) |

| `switch.first_floor_mev_dehumidifier` | Switch | ON/OFF |
| `switch.first_floor_mev_cooling` | Switch | ON/OFF |

## Risks & Mitigations

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|------------|
| Entity unavailable | Low | Medium | Variables have defaults |
| Race condition with stage transitions | Low | Low | Mode: restart handles this |
| Pump stuck ON | Low | High | Timer protection + manual override |
| Automation disabled | Medium | High | Periodic check in dashboard |
