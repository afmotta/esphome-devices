### packet_transport Provider Pattern

**Distribution Board YAML Structure:**

```yaml

  id: udp_packet_transport

packet_transport:
  providers:
    # Ground Floor Rooms (6)
    - soggiorno_room_sensor
    - cucina_room_sensor
    - bagno_room_sensor
    - anticamera_room_sensor
    - sala_pranzo_room_sensor

    # First Floor Rooms (8)
    - bagno_grande_room_sensor
    - camera_1_room_sensor
    - camera_2_room_sensor
    - camera_3_room_sensor
    - camera_4_room_sensor
    - camera_5_room_sensor
    - camera_6_room_sensor
    - corridoio_room_sensor

# UDP Receiver Sensors (add to sensor section)
sensor:
  # Soggiorno UDP Receivers
  - platform: packet_transport
    id: soggiorno_udp_temp
    provider: soggiorno_room_sensor
    internal: true
    
  - platform: packet_transport
    id: soggiorno_udp_humidity
    provider: soggiorno_room_sensor
    internal: true
  
  # Repeat for all 14 rooms (28 sensors total: 14 temp + 14 humidity)
```

### room_sensors.yaml v6 UDP Mode

**Component Interface (backward compatible):**

```yaml
# v5 Behavior (Epic 5 HA-only, default)
sensors: !include
  file: ../../room_sensors.yaml
  vars:
    zone_slug: "soggiorno"
    zone_name: "Soggiorno"
    ha_temperature_sensor_id: "sensor.room_soggiorno_temperature"
    # use_udp defaults to false → Epic 5 behavior preserved

# v6 Behavior (Epic 11 UDP with HA fallback)
sensors: !include
  file: ../../room_sensors.yaml
  vars:
    zone_slug: "soggiorno"
    zone_name: "Soggiorno"
    use_udp: true
    udp_sensor_id: "soggiorno_udp_temp"
    ha_temperature_sensor_id: "sensor.room_soggiorno_temperature"
```

**Component Implementation (room_sensors.yaml):**

```yaml
# components/room_sensors.yaml v6
defaults:
  use_udp: false  # Backward compatibility: default to Epic 5 HA-only

# Conditional delegation based on use_udp flag
{% if use_udp %}
  # Epic 11 UDP Mode: Delegate to 3-tier failover
  sensor_failover: !include
    file: sensor_failover_3tier.yaml
    vars:
      zone_slug: ${zone_slug}
      zone_name: ${zone_name}
      udp_sensor_id: ${udp_sensor_id}
      ha_temperature_sensor_id: ${ha_temperature_sensor_id}
      emergency_timeout: 180
{% else %}
  # Epic 5 HA-Only Mode: Direct HA sensor (existing behavior)
  sensor:
    - platform: homeassistant
      id: ${zone_slug}_ha_temperature
      entity_id: ${ha_temperature_sensor_id}
      # ... rest of Epic 5 config
{% endif %}
```

### Flash/RAM Budget Analysis

**Epic 9 Baseline (Supply Temp UDP):**
- Flash: ~67% (mixing group + supply temp broadcasts)
- RAM: ~12% (UDP packet buffers)

**Epic 11 Addition (14 Room Receivers):**
- 14 providers × ~200 bytes = 2.8 KB flash
- 14 temp sensors × ~300 bytes = 4.2 KB flash
- 14 humidity sensors × ~300 bytes = 4.2 KB flash
- UDP packet buffers: 14 × 64 bytes = 896 bytes RAM
- **Total overhead:** ~11.2 KB flash, ~1 KB RAM

**Estimated Final Usage:**
- Flash: ~70-72% (well below 80% limit)
- RAM: ~13-14% (well below 15% limit)
