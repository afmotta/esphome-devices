### Register Map

| Register | Address | Description                                           |
| -------- | ------- | ----------------------------------------------------- |
| 200      | 0x00C8  | Climate mode (0=off, 1=heat, 2=cool)                  |
| 300      | 0x012C  | Master heartbeat counter                              |
| 1000     | 0x03E8  | Slave health status (0=offline, 1=online, 2=degraded) |

- **Polling Interval:** 10 seconds
- **Sensor Timeout:** 30 seconds (3 missed reads)
- **Emergency Timeout:** 5 minutes (triggers shutdown)
