# Story 16.5: HA Helper Entities

**Epic:** 16 - First Floor MEV Intelligent Control  
**Date:** January 15, 2026  
**Status:** Ready  
**Story Points:** 1  
**Priority:** High

---

## User Story

As a **system integrator**,  
I want **all required Home Assistant helper entities created and configured**,  
So that **the MEV control automations have the state storage and timers they need**.

---

## Story Context

This story consolidates all HA helper entity creation into a single checklist. These helpers are referenced by Stories 16.2, 16.3, 16.4, and 16.6.

**Entity Types:**
- **Input Select:** State machine storage
- **Timers:** Pump protection timing
- **Template Sensors:** Diagnostic calculations

---

## Acceptance Criteria

### Functional Requirements

1. **Input Select Helpers:**
   - [ ] `input_select.mev_humidity_stage`
     - Options: normal, elevated, high, critical
     - Initial: normal

2. **Timer Helpers:**
   - [ ] `timer.mev_cooling_pump_min_on` (5 minutes)
   - [ ] `timer.mev_cooling_pump_min_off` (10 minutes)

3. **Template Sensors:**
   - [ ] `sensor.mev_air_quality_demand` (Story 16.2)
   - [ ] `sensor.mev_humidity_stage_demand` (fan demand based on stage)

### Technical Requirements

4. **All entities accessible in HA**
5. **Template sensors update correctly**
6. **Timers can be started/paused/cancelled**

---

## Technical Implementation

### Method 1: UI-Based Creation

Create via **Settings → Devices & Services → Helpers**:

| Entity | Type | Settings |
|--------|------|----------|
| `mev_humidity_stage` | Input Select | Options: normal, elevated, high, critical |
| `mev_cooling_pump_min_on` | Timer | Duration: 00:05:00 |
| `mev_cooling_pump_min_off` | Timer | Duration: 00:10:00 |

### Method 2: YAML-Based Creation

Create `packages/mev_helpers.yaml`:

```yaml
# Home Assistant Helpers for MEV Control
# Include in configuration.yaml via:
#   homeassistant:
#     packages:
#       mev_helpers: !include packages/mev_helpers.yaml

# ============================================
# Input Select - Humidity Cascade State Machine
# ============================================
input_select:
  mev_humidity_stage:
    name: "MEV Humidity Stage"
    options:
      - normal
      - elevated
      - high
      - critical
    initial: normal
    icon: mdi:water-percent

# ============================================
# Timers - Pump Protection
# ============================================
timer:
  mev_cooling_pump_min_on:
    name: "MEV Cooling Pump Min ON"
    duration: "00:05:00"
    icon: mdi:timer
    
  mev_cooling_pump_min_off:
    name: "MEV Cooling Pump Min OFF"
    duration: "00:10:00"
    icon: mdi:timer-off

# ============================================
# Template Sensors - Diagnostics
# ============================================
template:
  - sensor:
      # Air Quality → Fan Demand Mapping
      - name: "MEV Air Quality Demand"
        unique_id: mev_air_quality_demand
        unit_of_measurement: "%"
        device_class: power_factor
        icon: mdi:fan
        state: >
          {% set co2 = states('sensor.first_floor_max_co2') | float(600) %}
          {% set iaq = states('sensor.first_floor_max_iaq') | float(100) %}
          {% set co2_d = 90 if co2 > 1400 else 70 if co2 > 1000 else 50 if co2 > 800 else 35 if co2 > 600 else 20 %}
          {% set iaq_d = 90 if iaq > 350 else 70 if iaq > 250 else 50 if iaq > 150 else 35 if iaq > 100 else 20 %}
          {{ [co2_d, iaq_d] | max }}
        attributes:
          co2_level: "{{ states('sensor.first_floor_max_co2') | float(600) | round(0) }}"
          co2_demand: >
            {% set co2 = states('sensor.first_floor_max_co2') | float(600) %}
            {{ 90 if co2 > 1400 else 70 if co2 > 1000 else 50 if co2 > 800 else 35 if co2 > 600 else 20 }}
          iaq_level: "{{ states('sensor.first_floor_max_iaq') | float(100) | round(0) }}"
          iaq_demand: >
            {% set iaq = states('sensor.first_floor_max_iaq') | float(100) %}
            {{ 90 if iaq > 350 else 70 if iaq > 250 else 50 if iaq > 150 else 35 if iaq > 100 else 20 }}
          driving_signal: >
            {% set co2 = states('sensor.first_floor_max_co2') | float(600) %}
            {% set iaq = states('sensor.first_floor_max_iaq') | float(100) %}
            {% set co2_d = 90 if co2 > 1400 else 70 if co2 > 1000 else 50 if co2 > 800 else 35 if co2 > 600 else 20 %}
            {% set iaq_d = 90 if iaq > 350 else 70 if iaq > 250 else 50 if iaq > 150 else 35 if iaq > 100 else 20 %}
            {{ 'CO₂' if co2_d >= iaq_d else 'IAQ' }}

      # Humidity Stage → Fan Demand Mapping
      - name: "MEV Humidity Stage Demand"
        unique_id: mev_humidity_stage_demand
        unit_of_measurement: "%"
        device_class: power_factor
        icon: mdi:water-percent
        state: >
          {% set stage = states('input_select.mev_humidity_stage') %}
          {% set aq = states('sensor.mev_air_quality_demand') | int(20) %}
          {% if stage == 'critical' %}
            {{ [90, aq] | max }}
          {% elif stage == 'high' %}
            {{ [70, aq] | max }}
          {% elif stage == 'elevated' %}
            {{ [aq + 20, 100] | min }}
          {% else %}
            {{ aq }}
          {% endif %}
        attributes:
          humidity_stage: "{{ states('input_select.mev_humidity_stage') }}"
          aq_demand: "{{ states('sensor.mev_air_quality_demand') | int(20) }}"
          stage_boost: >
            {% set stage = states('input_select.mev_humidity_stage') %}
            {% if stage == 'critical' %}
              90% minimum
            {% elif stage == 'high' %}
              70% minimum
            {% elif stage == 'elevated' %}
              +20%
            {% else %}
              none
            {% endif %}

      # Final Commanded Fan Speed (for dashboard)
      - name: "MEV Final Fan Speed"
        unique_id: mev_final_fan_speed
        unit_of_measurement: "%"
        device_class: power_factor
        icon: mdi:fan
        state: "{{ states('sensor.mev_humidity_stage_demand') | int(20) }}"
```
