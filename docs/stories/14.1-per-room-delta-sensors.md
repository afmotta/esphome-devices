# Story 14.1: Per-Room Delta Sensors

**Epic:** 14 - Per-Room Fancoil Boost Control  
**Date:** December 26, 2025  
**Status:** Ready  
**Story Points:** 1  
**Priority:** High

---

## User Story

As a **system owner with independent room control**,  
I want **separate temperature delta sensors for each room**,  
So that **each room can independently evaluate its boost need**.

---

## Story Context

**Current State:** Epic 13 uses `ground_floor_max_delta` which calculates `max(soggiorno_delta, cucina_delta)` inline.

**Target State:** Separate sensors for each room:
- `soggiorno_temp_delta` - Soggiorno current temp - target temp
- `cucina_temp_delta` - Cucina current temp - target temp

---

## Acceptance Criteria

### Functional Requirements

1. **Soggiorno Delta Sensor:**
   - [ ] Create `soggiorno_temp_delta` template sensor
   - [ ] Calculate: `soggiorno_room_temp_abstracted - pid_radiant_soggiorno.target_temperature`
   - [ ] Positive value = room warmer than target (needs cooling)
   - [ ] Update interval: 30s

2. **Cucina Delta Sensor:**
   - [ ] Create `cucina_temp_delta` template sensor
   - [ ] Calculate: `cucina_room_temp_abstracted - pid_radiant_cucina.target_temperature`
   - [ ] Same characteristics as Soggiorno

3. **Sensor Characteristics:**
   - [ ] Unit: °C
   - [ ] Accuracy: 1 decimal
   - [ ] Device class: temperature
   - [ ] Icon: mdi:thermometer-lines

4. **Deprecation:**
   - [ ] Mark `ground_floor_max_delta` for removal in Story 14.3

---

## Technical Implementation

### Add to ground-floor.yaml or coordinator

```yaml
sensor:
  # Per-room temperature deltas for independent boost evaluation
  - platform: template
    id: soggiorno_temp_delta
    name: "Soggiorno Temperature Delta"
    unit_of_measurement: "°C"
    accuracy_decimals: 1
    device_class: "temperature"
    icon: "mdi:thermometer-lines"
    lambda: |-
      float temp = id(soggiorno_room_temp_abstracted).state;
      float target = id(pid_radiant_soggiorno).target_temperature;
      if (std::isnan(temp) || std::isnan(target)) return NAN;
      return temp - target;
    update_interval: 30s

  - platform: template
    id: cucina_temp_delta
    name: "Cucina Temperature Delta"
    unit_of_measurement: "°C"
    accuracy_decimals: 1
    device_class: "temperature"
    icon: "mdi:thermometer-lines"
    lambda: |-
      float temp = id(cucina_room_temp_abstracted).state;
      float target = id(pid_radiant_cucina).target_temperature;
      if (std::isnan(temp) || std::isnan(target)) return NAN;
      return temp - target;
    update_interval: 30s
```
