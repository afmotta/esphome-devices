- Protocol-header SSoT discipline held perfectly. All five on_multi_click handlers in button.yaml use can_id(CAT_INPUT, ${node_id}), button_payload(...), and EVT_* constants — no magic numbers or raw bit-shifts in lambdas. Epic 1's investment paid off as designed.
- Pragmatic platform adaptation over stubbornness:
  - 2.2: discovered ESPHome canbus.send: can_id: field is not templatable -> pivoted to id(can0).send_data(can_id(...), ...) inside a lambda.

- The NFR-2 size guard (if (x.size() < CAN_FRAME_SIZE) return;) was applied consistently and is now a codified reusable pattern.
- Code review triaged cleanly into Patch (fix now) / Decision / Defer (pre-existing), keeping each story's scope honest.
